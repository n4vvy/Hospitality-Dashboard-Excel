# Hospitality-Dashboard-Excel
Hospitality Dashboard - Excel

This project focuses on AtliQ Hotels, a fictional hotel chain operating seven properties across four major metropolitan cities — Delhi, Mumbai, Hyderabad, and Bangalore. The chain includes properties such as Atliq Grands, Atliq Exotica, Atliq City, Atliq Blu, Atliq Bay, Atliq Palace, and Atliq Seasons, catering to both Luxury and Business segments.
The primary objective of this project is to develop an interactive dashboard that enables the management of AtliQ Hotels to monitor key performance indicators, analyse trends, and derive actionable insights to improve business performance.
